Overview
-This project analyzes state-wise and age-wise Aadhaar enrolment trends in India using UIDAI public data.
-The goal is to identify key patterns and insights to support better enrolment planning and inclusive identity coverage.

Dataset
-Source: UIDAI (event.data.gov.in)
-Aggregated and anonymized public data

Tools Used
-Python, Pandas, Matplotlib, Seaborn

Key Analysis
-State-wise enrolment comparison
-Age-group distribution
-Year-wise enrolment trends
